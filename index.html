concrete5 Eclipse plugin

Improve your Eclipse + PDT experience when developing concrete5 projects.

Get Eclipse

  1. Go to the Eclipse Project Downloads
  2. Click the last build in the latest download section (4.7.1a at the time of writing)
  3. Download the Platform Runtime Binary suitable for you
  4. Extract the downloaded archive wherever you want

Install dependencies

  1. Open Eclipse
  2. Choose the Help → Install New Software... menu
  3. Hit the Manage button
  4. Add the concrete5 plugin update site
